# 🏠 Airbnb EDA – Statistical Computing

## 📌 Overview
This project focuses on **Exploratory Data Analysis (EDA)** of Airbnb listings data to uncover insights related to pricing, availability, location trends, and host behavior.

The analysis is performed using statistical computing techniques and visualization methods to better understand patterns in Airbnb listings and support data-driven decision-making.

---

## 🎯 Objectives
- Perform **data cleaning and preprocessing**
- Conduct **univariate and multivariate analysis**
- Identify key factors influencing:
  - Price
  - Availability
  - Popular locations
  - Review trends
- Generate **visual insights** using plots and statistical summaries

---

## 📊 Dataset
- **Source:** Airbnb NYC Open Dataset (Kaggle)
- ~49,000 listings with 16 features  
- Key attributes include:
  - Location (latitude, longitude, neighbourhood)
  - Price
  - Room type
  - Reviews & availability
  - Host-related features

## 🔍 Key Analysis Performed
- Data cleaning (handling missing values, duplicates)
- Distribution analysis of:
  - Prices
  - Room types
  - Neighborhood groups
- Correlation analysis between features
- Price variation across locations
- Availability trends
- Host activity and listing patterns

---

## 📈 Sample Insights
- Prices vary significantly across neighborhoods (e.g., Manhattan vs Brooklyn)
- Entire homes/apartments are more expensive than private/shared rooms
- High-review listings tend to have better availability
- Certain neighborhoods dominate listing density
